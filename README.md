# 🎓 Cloud-Based College Event Registration System

A simple and modern web application that allows students to register for college events such as Tech Fest, Cultural Fest, and Workshops.

---

## 🚀 Project Overview

This project demonstrates a full-stack web application using frontend, backend, and database integration. It is designed to be simple, efficient, and easily deployable on cloud platforms.

---

## ✨ Features

- User-friendly registration form  
- Multiple input fields (Name, Email, Phone, etc.)  
- Event selection (Tech Fest, Cultural Fest, Workshop)  
- Instant success message after registration  
- Data stored in database  
- Publicly accessible via web link  
- View all registrations page  

---

## 💻 Technologies Used

- **Frontend:** HTML, CSS  
- **Backend:** PHP  
- **Database:** SQLite  
- **Hosting:** InfinityFree  

---

## ⚙️ How It Works

1. User opens the website  
2. Fills the registration form  
3. Clicks "Register"  
4. Data is sent to PHP backend  
5. Data is stored in SQLite database (`events.db`)  
6. Success message is displayed
